# Hackatime Calculator
Actual estimated time calculator for coding projects!
Have you ever found yourself wanting to make multiple hours per day but you take time searching and learning, so it takes more actual time than you wanted? Here is a calculator to know how much actual time you would take to get your goal if you know you addinitonal time you take on searching! It is set to 15 mins if you didn't put a value
